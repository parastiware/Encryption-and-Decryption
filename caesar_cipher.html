Caesar Cipher

The Caesar Cipher is a substitution cipher where each letter in the plaintext is shifted a certain number of places down or up the alphabet.

For example, with a shift of 3, A would be replaced by D, B would become E, and so on. So the plaintext "HELLO" will be changed to "KHOOR" with the shift value 3 as shown below.
